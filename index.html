GUIDE

          
このツールの使い方とベストプラクティス

          

            SRT字幕を初めて扱う方でも迷わないよう、準備から書き出しまでの流れとコツをまとめました。
          

        


        

          

            
準備するもの

            
    
              
  • 音声ファイル（.wav）。収録時はピークが -1 dB 程度になるよう正規化すると精度が安定します。
    • 
              
  • 同じファイル名のテキスト（.txt）。1 行が 1 クリップになるよう改行で区切ります。
    • 
              
  • 動画編集ソフトのフレームレート情報。Premiere Pro や DaVinci Resolve で確認できます。
    • 
            

          


          

            
基本操作の流れ

            
    
              
  1. 「ファイルを選択」または「フォルダを選択」から、音声とテキストのペアを読み込みます。
    2. 
              
  2. フレームレートを指定し、必要に応じて自動調整やフレーム境界の丸め込みを設定します。
    3. 
              
  3. 「SRTを生成」を押すとブラウザ内で処理し、結果セクションに統計とダウンロードリンクが表示されます。
    4. 
              
  4. 生成した .srt を動画編集ソフトにインポートし、タイムラインに配置して微調整します。
    5. 
            

          


          

            
テキストファイル記述のポイント

            
    
              
  • 句読点や改行で視認性を高めると、字幕の分割位置が分かりやすくなります。
    • 
              
  • 顔文字や絵文字は SRT の互換性を損なう場合があるため、基本は全角／半角文字のみ推奨です。
    • 
              
  • 数字や記号を多用する場合は、テキストエンコーディングを UTF-8（BOM なし）にしてください。
    • 
            

          


          

            
よくある質問

            

              

                
Q. ブラウザにファイルはアップロードされますか？

                
A. いいえ。すべての処理はローカルで完結し、サーバーに送信されません。

              

              

                
Q. 字幕がずれるときは？

                

                  A. フレームレートの設定を動画と揃えたうえで、「合計時間に合わせて自動調整」を有効にすると、全体の長さを合わせやすくなります。
                

              

              

                
Q. 句読点の位置を後から変えたい

                
A. テキストファイルを修正して再生成すると、同じ順序で字幕が並び替えられます。

              

            

          


          

            
安全性とデータ取り扱い

            

              このツールはブラウザ内で処理が完結し、読み込んだ音声・テキストは外部に送信されません。実務で利用する場合は、
              機密データが含まれていないか を確認したうえで操作してください。
            

            

              生成した SRT は UTF-8 で出力されます。Premiere Pro、DaVinci Resolve、CapCut など主要な編集ソフトで読み込めますが、環境依存文字は避けることを推奨します。
            

            

              Web UI が反応しない場合はブラウザのシークレットモードを解除し、最新バージョンの Chrome / Edge をお使いください。
            

          

        

      

    


    

      

        同名の .wav と .txt ファイルをペアにして処理します。ファイル名は自動的に昇順で並び替えられます。
